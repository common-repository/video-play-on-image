Video Play On Image is a Elementor Addon to display video on Elementor website. You can add a video thumbnel on a salf hosted video (YouTube video URL curently not avalable).

== Description ==

👉 Elementor plugin required 👈


== 🏆 Features of The Plugin ==

* **Elementor Drag and drop widget**
* **Upload custom thumbnail**
* **Upliad video on media**
* **You can change Play Button Color**
* **You can ON/OFF video sound**
* **Video loop ON/OFF Button**
* **You can change Hight width of image and Video also**
* **You can adjust reonded corner video**
* **Image & video same frame**

== Easy Steps to configer this Plugin ==

Just install the plugin and activate it.

1. Edit page with Elementor page builder.

    * **Step1: search for video play on image in the widget serch box or you can find it on BASIC tab**
    * **Step2: Drag & drop your page**
    * **Step3: Upload thumbnail & video**
    * **Step4: Change Play Button Color according to template button color**
    * **Step5: Mute video sound if required**
    * **Step6: loop video ON/OFF**
    * **Step5: Adjust Size & Shape**


== Installation ==

1. Download `video-play-on-image` zip file
2. Upload the 'video-play-on-image' zip and its contents into the `wp-content/plugins/` directory of your WordPress installation
3. Activate `Video Play On Image` from Plugins menu in WordPress.


= Requirements =
* **WordPress version:** >= 5.0
* **PHP version:** >= 7.2
